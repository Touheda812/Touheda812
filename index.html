My MTA Internship Projects

During my internship with the Metropolitan Transportation Authority (MTA), I developed interactive web tools to improve accessibility and efficiency for commuters. Below are the projects I designed and deployed.

MTA Toll Calculator

A responsive web application that calculates toll costs for various MTA bridges and tunnels. Built with HTML, CSS, and JavaScript, the tool makes it easier for drivers to plan trips.

Live Demo GitHub

CBDTP Calculator

A web-based calculator designed for the Central Business District Tolling Program (CBDTP). The app helps commuters understand potential tolling impacts in NYC's congestion pricing zones.

Live Demo GitHub
